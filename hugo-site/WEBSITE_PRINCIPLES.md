# Website Principles

This site follows the academic website intake prompt:

- Hugo source lives in `hugo-site/`.
- Blox is imported as a Go module and should be vendored with `hugo mod vendor`.
- Pagefind is built after Hugo and before GitHub Pages artifact upload.
- Public copy should read like an academic website, not a migration log.
- The homepage intro and full bio must be distinct.
- The navbar brand is the owner's full name and links home.
- All placeholders must be removed before publication.
